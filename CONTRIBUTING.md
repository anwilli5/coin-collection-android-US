# How to contribute

If you want to make a contribution to the repository, follow these steps.
Please follow the [Code of Conduct](https://github.com/anwilli5/coin-collection-android-US/blob/main/CODE_OF_CONDUCT.md) and keep it in mind while contributing.

1. **Fork this repository** to your own GitHub account by clicking the Fork button on the top right corner.

2. **Clone the repository**

   Clone this repository to your local system. Copy the link from your remote repo and run the following command:

   ```bash
   git clone https://github.com/<username>/coin-collection-android-US.git
   ```

3. **Make your changes/contributions**

   Make your changes/contributions in your local repository.

4. **Create a new branch**

   Before committing your changes, create a new branch for the feature/bug you are working on and check out that branch:

   ```bash
   git checkout -b <feature/bug_branch_name>
   ```

5. **Stage your changes**

   Stage your changes:

   ```bash
   git add .
   ```

6. **Commit your changes**

   Commit your changes and add a commit message:

   ```bash
   git commit -m 'describe your changes here'
   ```

7. **Push the changes**

   Push these changes to your remote repo:

   ```bash
   git push origin <feature_branch_name>
   ```

8. **Raise a Pull Request**

   After pushing the changes, come back to this repository and reload the page. You will find an option that says **"Compare and Pull request"**. Click on that, give the title and description of the changes you made, and click **Create pull request**.

   If you don't see that option, go to the [Pull Requests page](https://github.com/anwilli5/coin-collection-android-US/pulls), click **New pull request**, select **compare across forks**, choose your repo and branch against the base branch, and click **Create pull request**.

**That's it! You have successfully made your contribution.**

If you want to request a new feature or report a bug, feel free to [report an issue](https://github.com/anwilli5/coin-collection-android-US/issues/new/choose).
